# フィルターの作成{#creating-filters}

Adobe Campaignで提供される組み込みの受信者テーブルと同様に、新しい受信者テーブルには、事前に定義されたフィルターのバッチが送信される場合があります。

これらのフィルターは、受信者のセグメントと同じ機能（パラメーター入力フォーム、フォルダーなどを使用）を持つターゲット選択ウィンドウで使用できます。

1. **[!UICONTROL 管理/設定/定義済みフィルター]** ノードに移動します。
1. 新しいフィルターを作成します。
1. フィルターの&#x200B;**[!UICONTROL ラベル]**&#x200B;を入力し、**[!UICONTROL ドキュメントタイプ]** フィールドで外部受信者テーブルと一致するスキーマを選択します。
1. スキーマのフィールドに基づいて&#x200B;**[!UICONTROL フィルター条件]**&#x200B;を作成します。
1. フィルターを保存します。

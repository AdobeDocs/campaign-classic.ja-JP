# 分岐{#fork}

分岐を使用して、複数のアクティビティを並行して開始できます。

グラフィックオブジェクトをダブルクリックして、アウトバウンドトランジションの数を定義したり、新規トランジションを作成したり、選択したトランジションのラベルを変更したりします。

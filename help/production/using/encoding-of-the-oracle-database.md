# Oracleデータベースのエンコード{#encoding-of-the-oracle-database}

Adobe Campaignでは、デフォルトのOracleエンコーディングが使用されます。 異なる互換性を使用すると、非互換性が発生する場合があります。 詳しくは、[この節](../../installation/using/database.md#oracle)を参照してください。

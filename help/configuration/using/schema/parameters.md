![](../../../assets/v7-only.svg)

## コンテンツモデル {#content-model-13}

パラメータ：==param

## 属性 {#attributes-13}

なし

## 親 {#parents-13}

`<method>`

## 子 {#children-13}

`<param>`

## 説明 {#description-13}

この要素は `<parameter>` 要素のグループを定義します。

## 使用と使用のコンテキスト {#use-and-context-of-use-8}

この要素は必須です。`<method>` 要素の 1 つの `<param>` 子要素の場合でも同様です。

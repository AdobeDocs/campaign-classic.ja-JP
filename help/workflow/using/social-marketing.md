# ソーシャルマーケティング{#social-marketing}

以下に説明するワークフローは、デフォルトで&#x200B;**ソーシャルマーケティング**&#x200B;モジュールと共にインストールされます。このモジュールについて詳しくは、この[節](../../social/using/starting-workflows.md)を参照してください。

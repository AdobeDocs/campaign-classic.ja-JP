# Oracle データベースのエンコード{#encoding-of-the-oracle-database}



Adobe Campaignでは、デフォルトのOracleエンコーディングを使用します。

異なるものを使用すると、非互換性が発生する場合があります。 詳しくは、[この節](../../installation/using/database.md#oracle)を参照してください。

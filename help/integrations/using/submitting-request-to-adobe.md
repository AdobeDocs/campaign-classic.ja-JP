# アドビへのリクエストの送信{#submitting-request-to-adobe}

>[!NOTE]
>
>v7 ユーザーが手順を続けるには、お使いのシステムで Adobe Identity Management System（IMS）が有効になっていることを確認してください。 [詳細情報](../../integrations/using/about-adobe-id.md)。

お客様の組織で Adobe Identity Management サービス（IMS）が有効になると、Campaign と [Experience Cloud のオーディエンス](https://experienceleague.adobe.com/ja/docs/core-services/interface/services/audiences/overview){target="_blank"}を統合するプロビジョニングをリクエストできます。これを行うには、アドビ担当者に問い合わせて、次の情報を入力します。

# メソッド要素{#methods--element}

## コンテンツモデル{#content-model-11}

methods:==method

## 属性 {#attributes-11}

なし

## 親{#parents-11}

`<srcschema>`

## 子 {#children-11}

method

## 説明 {#description-11}

この要素を使用すると、`<method>`要素を定義できます。 メソッドを宣言する場合は必須です。

## 属性の説明{#attribute-description-11}

この要素には属性がありません。

## 例 {#examples-8}

```
<methods async="true"
...// definition of one or more <method
</methods>
```

# 復元{#restoration}



クリーンサーバーでは、復元手順は次のようになります。

* インストールおよび構成済みのオペレーティングシステム（ネットワーク）上で、
* サードパーティアプリケーションのインストール：Web サーバー、JDK（必要に応じて）
* ソースシステムと同じビルドでAdobe Campaignバイナリをインストールする。
* 設定ファイル、トラッキングログ、リダイレクションファイルのコピー
* データベースを作成して再構築します。
* Adobe Campaignを起動します。

詳しくは、 **インストールガイド**.

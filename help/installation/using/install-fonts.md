# フォントのインストール {#installing-fonts}



Campaign アプリケーションサーバーに新しいフォントをインストールする方法を次に示します。

新しいフォントは、MTA プロセスが実行されるすべてのコンテナの`/usr/local/share/` フォルダーに配置できます。
